# 数据中心

## 简介

数据中心将所有采集或导入的数据汇集到一起，进行统一的数据融合与存储。它为企业提供一个 **_统一的、持久的、可共享_** 的客户数据库，解决全渠道、多触点、多业务系统、线上线下的复杂业务中所面临的数据孤岛问题 。
